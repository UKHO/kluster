LoggerClass¶

class HSTB.kluster.fqpr_intelligence.LoggerClass(silent=False, logger=None)¶

Bases: object

Basic class for logging. Include a logging.logger instance to use that, or set silent to true to disable print messages entirely. Use of Logger will trump silent.

Methods Summary

print_msg(msg[, loglvl])
